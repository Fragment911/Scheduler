Три роли пользователя:
	Обычный - создает, изменяет, удаляет списки задач; создает, изменяет, удаляет задачи внутри списков
	Модератор - вышеперечисленное + может просматривать списки и задачи всех пользователей
	Администратор - вышеперечисленное + может повышать и понижать роли других пользователей и удалять аккаунты
